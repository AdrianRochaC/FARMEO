### For installation in the desk

1. Install node.js in the device
2. Install npm global
3. Clone the repository
4. Install XAMPP
5. Import the database
6. Create a .env in the root of the project and create .env in the backend
7. Execute the commando "npm install"
8. Execute the frontend with the command "npm run dev"
9. Execute the backend with the command "cd backend" and "node server.js"
